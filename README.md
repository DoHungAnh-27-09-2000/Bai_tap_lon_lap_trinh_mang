## Ứng dụng destop chat và chơi game online dùng Spring Data JPA và Java socket, Java Swing. "chat-server" mở bằng intellij, "chat-client" mở bằng netbeans
## Công nghệ sử dụng
- Spring Data JPA
- Java Socket
- Java Multi Thread
- Java Swing
- MYSQL
